DeepSeek API的注册步骤：



🚀 DeepSeek API 注册详细步骤



第一步：访问官网



打开 DeepSeek 官方平台：platform.deepseek.com



第二步：注册账号



1\. 点击页面上的“注册”或“Sign Up”按钮

2\. 选择注册方式：

   • 邮箱注册：输入电子邮箱，接收验证码



   • 手机号注册：输入手机号，接收短信验证码



   • 第三方账号：可能支持GitHub、Google等快速登录



第三步：验证账号



• 检查邮箱或手机，输入收到的验证码



• 设置登录密码



第四步：登录控制台



• 使用刚注册的账号登录



• 进入开发者控制台（Dashboard）



第五步：获取API密钥



1\. 在控制台找到“API Keys”或“API密钥” section

2\. 点击“Create New API Key”或“生成新密钥”

3\. 为密钥设置一个识别名称（如“我的项目”）

4\. 重要：复制生成的API Key并安全保存（只显示一次！）



第六步：查看文档和额度



• 查看API文档了解使用方法



• 在控制台查看免费额度使用情况（每月100万tokens）



💡 小贴士



• 注册过程完全免费，无需绑定支付方式



• API Key是访问凭证，请妥善保管不要泄露



• 免费额度足够个人项目使用
